My implementations for [AdventOfCode](https://adventofcode.com/)


# 2024 How to

Create folder for each day
`mkdir Day08`

Go into
`cd Day08`

Create outputfolder
`mkdir output`

Create and manipulate main.cpp
`touch main.cpp`

Compile (Windows)
`cl.exe /Zi /EHsc /nologo /Feoutput\main.exe main.cpp`

Run
`.\output\main.exe`
